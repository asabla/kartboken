# Changelog

All notable Kartboken changes are recorded here. Versions follow semantic versioning.

## 1.0.0 - 2026-07-16

### Added

- A static map-first guide with an accessible, source-backed place list.
- Five manually reviewed restaurant and café recommendations from the originating Reddit thread.
- Swedish-aware deterministic search, category filters, journey-value filters, map/list selection, and shareable URL state.
- A versioned YAML catalog contract with recommendation, official, and geodata provenance.
- Lazy MapLibre loading with a configurable OpenFreeMap-compatible style URL.
- Static-host and hardened Caddy container deployment options.
- Catalog validation, unit tests, fuzzy properties, desktop/mobile/no-JavaScript browser tests, container verification, and browser bundle budgets in CI.
- Repository guidance and focused skills for curation, verification, and durable agent documentation.
